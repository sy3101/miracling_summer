미라클링 썸머스쿨 탭북 배포 파일입니다.

로컬 확인:
1. 압축을 풉니다.
2. 로컬_미리보기_실행.bat을 실행합니다.
3. 브라우저에서 http://localhost:5173 이 열립니다.

Vercel 배포:
1. 이 폴더 전체를 GitHub 저장소에 업로드합니다.
2. Vercel에서 Add New > Project > 해당 GitHub 저장소 Import를 누릅니다.
3. Framework Preset은 Other로 두고 Deploy를 누릅니다.

주의:
- index.html, vercel.json, assets 폴더가 같은 위치에 있어야 합니다.
- assets 폴더 안에 1.jpg부터 8.jpg까지 있어야 합니다.
